# 🎓 Student Course Recommendation System

A Machine Learning–based application that recommends suitable courses for students based on academic profile, skills, interests, CGPA, and coding level.

## 🚀 Features
- Personalized course recommendations using Random Forest Classifier
- Skill-based and profile-based prediction logic
- Interactive Streamlit dashboard
- Displays recommended courses, skills gained, and career paths
- Supports students with and without prior coding experience

## 🛠 Tech Stack
- Python
- Machine Learning
- Scikit-learn
- Pandas
- Streamlit

## 📊 Machine Learning Approach
- Supervised learning (Classification)
- Feature engineering and label encoding
- MultiLabelBinarizer for skill encoding
- Probability-based ranking for top recommendations
